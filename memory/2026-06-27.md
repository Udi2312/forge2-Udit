### CRITICAL: Ticket model needs `slaStatus()` method
The TicketController calls `$ticket->slaStatus()` but this method has NOT been added to the Ticket model yet. Must add:
```php
public function slaStatus(): array
{
    $policy = \App\Models\SlaPolicy::where('organization_id', $this->organization_id)
        ->where('priority', $this->priority)
        ->where('is_active', true)
        ->first();

    if (!$policy) return ['status' => 'none'];

    $created = $this->created_at;
    $responseDeadline = $created->addMinutes($policy->response_time_minutes);
    $resolutionDeadline = $created->addMinutes($policy->resolution_time_minutes);
    $now = now();

    if (in_array($this->status, ['resolved', 'closed'])) return ['status' => 'met', 'response_deadline' => $responseDeadline, 'resolution_deadline' => $resolutionDeadline];
    if ($now > $resolutionDeadline) return ['status' => 'breached', 'response_deadline' => $responseDeadline, 'resolution_deadline' => $resolutionDeadline];
    // warning when past response deadline but not resolution
    if ($now > $responseDeadline) return ['status' => 'warning', 'response_deadline' => $responseDeadline, 'resolution_deadline' => $resolutionDeadline];
    return ['status' => 'on_track', 'response_deadline' => $responseDeadline, 'resolution_deadline' => $resolutionDeadline];
}
```
